# E-Commerce App

A full-stack e-commerce application built with React, Express, MongoDB, and Redis.

## Features

- User registration and login
- JWT-based authentication
- Product listing and product details
- Basket management
- Order creation
- Admin panel for product management

## Tech Stack

### Frontend
- React
- React Router
- Chakra UI
- Axios

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
- Redis
- JWT
